# 📊 Gold Price Analysis 

This project analyzes gold futures price data using Python.  
It uses data from Yahoo Finance and performs basic financial analysis such as returns and volatility.

---

## 📌 Project Features

- Download historical gold price data (GC=F)
- Calculate basic statistics (mean, max, min)
- Compute log returns
- Calculate annualized volatility
- Visualize gold price trend
- Save cleaned data to CSV

---

## 📦 Libraries Used

- yfinance
- pandas
- numpy
- matplotlib

---

##  How to Run

1. Install required libraries:

```bash
pip install yfinance pandas numpy matplotlib
```

2. Run the script:
python gold_analysis.py

---

## Output
- Average Gold Price
- Maximum & Minimum Price
- Annualized Volatility
- Price Chart
